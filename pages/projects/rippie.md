# Rippie

## Project

A security tool to keep secret messages to be delivered in case of problems. Here everyone can create new messages, set an email to receive and a scheduler to be a life proof. 

## Tecnologies

Base

- Next.js
- Mailtrap
- MongoDB atlas

Deployment:

- Vercel
